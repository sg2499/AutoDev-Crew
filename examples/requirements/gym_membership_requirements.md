# Sample Requirements: Gym Membership Manager

Build a gym membership management app.

Each member should include member ID, name, phone, email, membership plan, start date, expiry date, payment status, trainer assigned, attendance count, and notes.

The system should add members, renew memberships, update payment status, assign trainer, mark attendance, list all members, filter by plan, filter by payment status, identify expiring memberships, identify unpaid members, and generate membership summary.

Validate email, phone, positive plan duration, valid date ranges, supported plan/payment status, and non-existing member operations.

The Gradio app should display members, expiring memberships, and unpaid members as tables. Membership summary should be Markdown and summary metrics.
