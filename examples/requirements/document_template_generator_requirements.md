# Sample Requirements: Document Template Generator

Build a document template generator app.

The system should allow users to define templates with template ID, template name, document type, placeholder fields, template text, created date, and notes.

The system should create templates, update templates, delete templates, list templates, filter by document type, generate a document by filling placeholders, validate missing placeholders, and produce a final document preview.

Prevent empty template names, duplicate template IDs, missing required placeholder values, unsupported document types, and updates to non-existing templates.

The Gradio app should display template lists as tables. Generated documents should be shown as readable Markdown or downloadable text files if requested.
