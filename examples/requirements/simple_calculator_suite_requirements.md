# Sample Requirements: Business Calculator Suite

Build a business calculator suite.

The system should provide calculators for profit margin, markup percentage, GST/tax amount, discount amount, break-even quantity, and simple interest.

Each calculator should validate numeric inputs and prevent invalid values such as negative selling price, zero cost price where division is required, invalid tax percentages, and negative quantities.

The backend should expose clear calculation methods and return both the numeric result and a short explanation of the formula used.

The Gradio app should use number inputs and display results as readable Markdown or numeric outputs. It should not show raw JSON unless explicitly requested.
