# Sample Requirements: Lesson Plan Manager

Build a lesson plan management app for teachers.

Each lesson plan should include lesson ID, subject, grade, topic, learning objectives, materials required, activity plan, assessment method, planned date, status, and notes.

The system should create lesson plans, update status, edit lesson details, delete plans, list lesson plans, filter by subject, filter by grade, filter by status, show upcoming lessons, and generate weekly teaching plan summary.

Prevent empty topics/objectives, invalid dates, unsupported statuses, duplicate lesson IDs, and updates to non-existing lessons.

The Gradio app should show lesson plans, upcoming lessons, and filtered plans as tables. Weekly summaries should be readable Markdown.
