# Sample Requirements: Worksheet Question Bank Manager

Build a worksheet question bank manager.

Each question should include question ID, subject, grade, topic, question text, question type, difficulty, marks, correct answer, and explanation.

The system should add questions, update questions, delete questions, list questions, filter by subject/grade/topic/difficulty, calculate total marks, generate a worksheet from selected filters, and generate an answer key.

Prevent empty question text, unsupported question types/difficulties, negative marks, duplicate IDs, and updates to non-existing questions.

The Gradio app should display question banks and generated worksheets as tables. Answer keys and worksheet summaries should be readable Markdown.
