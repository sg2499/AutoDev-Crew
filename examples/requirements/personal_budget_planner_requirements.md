# Sample Requirements: Personal Budget Planner

Build a personal budget planning app.

Each budget item should include item ID, category, planned amount, actual amount, month, year, status, and notes.

The system should add budget items, update actual amount, list budget items, filter by category, filter by month/year, calculate total planned, calculate total actual, calculate variance, identify overspent categories, and generate monthly budget report.

Prevent negative amounts, invalid months/years, empty categories, duplicate category-month entries, and updates to non-existing items.

The Gradio app should show budget records and overspent categories as tables. Monthly budget reports should be readable Markdown metrics and tables.
