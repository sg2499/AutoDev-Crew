# Sample Requirements: Property Rental Manager

Build a property rental management app.

Each rental record should include property ID, tenant name, tenant phone, property address, rent amount, lease start date, lease end date, payment status, security deposit, and notes.

The system should add rentals, update payment status, renew lease, terminate rental, list rentals, filter by payment status, identify lease expiries, calculate monthly rent receivable, calculate pending rent, and generate rental summary.

Prevent negative rent/deposit, invalid lease date ranges, empty tenant names, unsupported payment statuses, and updates to non-existing rentals.

The Gradio app should show rentals, pending rent records, and lease expiry records as tables. Rental summaries should be Markdown metrics.
