# Sample Requirements: Student Gradebook Manager

Build a student gradebook app for teachers.

The system should allow users to add students, record marks for assignments, quizzes, and exams, calculate total marks, calculate percentage, assign grade letters, list student performance, filter students by grade, and identify students needing improvement.

Each student record should include student ID, student name, class/section, subject, assignment marks, quiz marks, exam marks, total marks, percentage, grade, and teacher comments.

The system should prevent negative marks, marks above configured maximum, empty student names, duplicate student IDs, and updates to non-existing students.

The Gradio app should show gradebook records, filtered records, and low-performing students as tables. Performance summaries should be displayed as readable reports.
