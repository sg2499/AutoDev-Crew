# Sample Requirements: Personal Habit Tracker

Build a personal habit tracking app.

Each habit should include habit ID, habit name, category, target frequency, start date, status, streak count, completion dates, and notes.

The system should add habits, mark habit completed for a date, update habit status, delete habits, list habits, filter by category, calculate streaks, identify missed habits, calculate completion rate, and generate weekly habit summary.

Prevent empty habit names, unsupported frequencies/statuses, invalid dates, duplicate habit IDs, and updates to non-existing habits.

The Gradio app should show habit lists, missed habits, and completion history as tables. Weekly summaries should be readable Markdown reports.
