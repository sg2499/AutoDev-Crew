# Sample Requirements: Social Media Idea Bank

Build a social media idea bank app.

Each idea should include idea ID, title, target audience, platform, content format, hook, key message, status, priority, and notes.

The system should create ideas, update status, update priority, list ideas, filter by platform, filter by status, search ideas by title/audience, identify high-priority ideas, and generate a content ideation summary.

Prevent empty titles/hooks, unsupported platforms/statuses/priorities, duplicate IDs, and non-existing idea updates.

The Gradio app should show idea lists, search results, and high-priority ideas as tables. Ideation summaries should be readable Markdown.
