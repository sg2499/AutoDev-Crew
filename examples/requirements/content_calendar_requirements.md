# Sample Requirements: Content Calendar Planner

Build a content calendar planning app for creators.

Each content item should include content ID, title, platform, content type, publish date, status, owner, caption draft, hashtags, and notes.

The system should add content items, update status, reschedule posts, delete items, list calendar items, filter by platform, filter by status, show upcoming posts, calculate content counts by platform, and generate weekly content summary.

Prevent invalid dates, empty titles, unsupported platforms/statuses/content types, duplicate IDs, and non-existing item updates.

The Gradio app should show calendar items, upcoming posts, and filtered content as tables. Weekly summaries should be Markdown reports.
