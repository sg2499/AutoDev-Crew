# Sample Requirements: Task Manager AI

Build a task manager where users can:

- create tasks with title, description, priority, and due date,
- update task status,
- list pending and completed tasks,
- mark tasks completed,
- delete tasks,
- view simple task statistics,
- prevent invalid task updates.

The app should include a Python backend module, a Gradio interface, unit tests, documentation, security notes, and production-readiness assets.
