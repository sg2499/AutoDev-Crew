# Sample Requirements: Job Application Tracker

Build a job application tracking app for job seekers.

Each application should include application ID, company name, job title, job location, application date, job portal/source, status, recruiter contact, interview date, salary range, and notes.

The system should add applications, update status, update interview date, delete applications, list all applications, filter by status, filter by company, identify upcoming interviews, calculate applications by status, and generate job search summary.

Prevent empty company/job titles, invalid dates, unsupported statuses, duplicate IDs, and updates to non-existing applications.

The Gradio app should show application lists, filtered results, and upcoming interviews as tables. Job search summaries should be readable Markdown.
