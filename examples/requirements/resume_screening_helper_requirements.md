# Sample Requirements: Resume Screening Helper

Build a resume screening helper for recruiters.

The system should allow users to add candidate profiles with candidate ID, name, email, role, skills, experience years, education, resume summary, screening score, status, and notes.

The system should calculate a basic score by matching candidate skills against job-required skills, list candidates, filter by role, filter by status, identify top candidates, update candidate status, and generate a screening report.

Prevent invalid emails, negative experience, empty candidate names, empty required skills, unsupported statuses, and updates to non-existing candidates.

The Gradio app should show candidate lists, top candidates, and filtered candidates as tables. Screening reports and match explanations should be readable Markdown.
