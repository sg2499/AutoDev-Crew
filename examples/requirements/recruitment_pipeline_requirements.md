# Sample Requirements: Recruitment Pipeline Manager

Build a recruitment pipeline management app for HR teams.

Each candidate should include candidate ID, candidate name, email, phone, role applied, experience years, current stage, interview date, recruiter, rating, and notes.

The system should add candidates, update stage, schedule interview, update rating, reject/select candidate, list candidates, filter by role, filter by stage, identify upcoming interviews, calculate candidates by stage, and generate hiring pipeline summary.

Validate email, phone, non-negative experience, rating range, supported stages, and non-existing candidate operations.

The Gradio app should show candidate lists, stage filters, and upcoming interviews as tables. Pipeline summary should be Markdown and summary tables.
