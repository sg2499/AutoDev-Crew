# Sample Requirements: Nonprofit Donation Tracker

Build a donation tracking app for a nonprofit.

Each donation should include donation ID, donor name, donor email, donation amount, donation date, campaign name, payment method, receipt status, and notes.

The system should add donations, update receipt status, delete donations, list donations, filter by campaign, filter by payment method, calculate total donations, calculate donations by campaign, identify top donors, and generate a campaign summary.

Prevent invalid email, zero/negative donations, empty donor names, unsupported receipt statuses, and updates to non-existing donations.

The Gradio app should show donations, filtered donations, and top donors as tables. Donation summaries should be readable Markdown reports.
