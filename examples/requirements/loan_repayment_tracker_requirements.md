# Sample Requirements: Loan Repayment Tracker

Build a loan repayment tracking app.

Each loan should include loan ID, borrower name, principal amount, interest rate, monthly installment, start date, due date, amount paid, outstanding amount, status, and notes.

The system should add loans, record repayments, update status, list loans, filter by status, identify overdue loans, calculate total outstanding, calculate total repaid, and generate repayment summary.

Prevent negative amounts/rates, repayments greater than outstanding unless closing, invalid dates, unsupported statuses, and updates to non-existing loans.

The Gradio app should show loan records and overdue loans as tables. Repayment summaries should be Markdown metrics.
