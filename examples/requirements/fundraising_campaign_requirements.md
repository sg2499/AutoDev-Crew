# Sample Requirements: Fundraising Campaign Manager

Build a fundraising campaign management app.

Each campaign should include campaign ID, campaign name, target amount, collected amount, start date, end date, status, owner, and notes.

The system should create campaigns, update collected amount, update status, list campaigns, filter by status, identify campaigns behind target, calculate progress percentage, calculate total funds raised, and generate campaign performance summary.

Prevent negative targets, collected amount above target unless allowed by notes, invalid date ranges, unsupported statuses, and duplicate IDs.

The Gradio app should show campaigns and behind-target campaigns as tables. Performance summaries should be Markdown metrics.
