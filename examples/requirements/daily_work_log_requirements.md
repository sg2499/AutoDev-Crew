# Sample Requirements: Daily Work Log Manager

Build a daily work log management app.

Each log should include log ID, employee name, date, project name, task description, hours spent, task status, blockers, and notes.

The system should add logs, update status, delete logs, list logs, filter by employee, filter by project, filter by date range, calculate total hours, identify blocked tasks, and generate weekly work summary.

Prevent negative hours, hours above 24 per day, invalid date ranges, empty task descriptions, unsupported statuses, and updates to non-existing logs.

The Gradio app should display logs, blocked tasks, and filtered logs as tables. Weekly summaries should be Markdown reports.
