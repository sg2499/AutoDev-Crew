# Sample Requirements: Abacus Practice Sheet Generator

Build an abacus practice sheet generator for children.

The system should allow users to create practice configurations with configuration ID, level, concept type, number of questions, number of digits, operations allowed, time limit, and marks per question.

The system should generate practice questions, list generated sheets, filter by level/concept, calculate total marks, generate answer keys, and create a practice sheet summary.

Concept types should include direct addition/subtraction, five-complement practice, ten-complement practice, mixed practice, multiplication, and division.

Prevent invalid levels, zero/negative question counts, unsupported concept types, unsupported operations, and invalid digit ranges.

The Gradio app should show generated questions as tables. Answer keys and practice summaries should be readable Markdown.
