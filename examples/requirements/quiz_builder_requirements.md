Build a quiz builder and practice app for teachers.

The system should allow a teacher to create questions with subject, topic, question text, four options, correct answer, difficulty, and marks.

The system should list questions, filter by subject or difficulty, calculate total marks, generate a simple practice quiz, and show an answer key.

The Gradio app should display question banks and generated quiz questions as tables, while the answer key and quiz summary should be shown as readable markdown.
