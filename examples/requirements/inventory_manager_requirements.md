Build an inventory management app for a small retail shop.

The system should allow users to add products with SKU, product name, category, quantity, reorder level, supplier name, and unit price.

The system should allow users to update stock quantity, delete products, list all products, filter products by category, show low-stock products, calculate total inventory value, and generate a reorder report.

The Gradio app should show product and low-stock records as tables. Summary outputs should be readable reports, not raw JSON.
