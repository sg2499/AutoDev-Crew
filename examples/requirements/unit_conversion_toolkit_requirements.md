# Sample Requirements: Unit Conversion Toolkit

Build a unit conversion toolkit.

The system should support conversions for length, weight, temperature, volume, and area.

The backend should expose methods for converting between common units such as meters/feet/kilometers/miles, kilograms/pounds/grams, Celsius/Fahrenheit/Kelvin, liters/milliliters/gallons, and square feet/square meters.

The system should validate supported units, numeric values, and conversion categories. It should return converted values and a short explanation of the conversion.

The Gradio app should use dropdowns and number inputs. Results should be displayed as numbers and readable Markdown explanations, not tables unless conversion history is requested.
