# Sample Requirements: Weekly Meal Planner

Build a weekly meal planning app.

Each meal plan entry should include entry ID, day, meal type, recipe name, ingredients, calories, preparation time, dietary tag, and notes.

The system should add meals, update meals, delete meals, list weekly plan, filter by day, filter by dietary tag, calculate total calories by day, generate shopping list from ingredients, and create weekly meal summary.

Prevent unsupported days/meal types, negative calories, empty recipe names, duplicate day-meal entries, and updates to non-existing entries.

The Gradio app should display meal plans and shopping lists as tables. Weekly summaries should be readable Markdown.
