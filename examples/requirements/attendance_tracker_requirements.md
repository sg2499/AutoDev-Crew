# Sample Requirements: Employee Attendance Tracker

Build an employee attendance tracking app.

Each attendance record should include record ID, employee ID, employee name, department, date, check-in time, check-out time, status, and notes.

The system should allow users to add attendance records, update check-out time, mark absence, list records, filter by employee, filter by department, filter by date range, calculate total present days, calculate absences, identify late check-ins, and generate monthly attendance summary.

Prevent invalid dates, check-out before check-in, unsupported statuses, duplicate employee-date records, and updates to non-existing records.

The Gradio app should show attendance records, filtered results, and late check-ins as tables. Monthly summaries should be Markdown reports and summary tables.
