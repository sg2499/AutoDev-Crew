🚀 I built AutoDev Crew — an Agentic AI Software Engineering Platform powered by CrewAI and OpenAI GPT-4o.

Instead of using a single prompt to generate code, AutoDev Crew works like a compact AI engineering team. It takes natural-language software requirements and routes them through specialized agents for product specification, architecture, backend development, Gradio frontend generation, unit testing, code review, security review, and documentation.

What makes it more than a basic code generator:

✅ Multi-agent SDLC workflow  
✅ GPT-4o powered agent orchestration  
✅ Dynamic requirement intake  
✅ Gradio dashboard  
✅ Generated backend + frontend + tests  
✅ Automated syntax validation and unittest execution  
✅ Optional GPT-4o repair loop  
✅ Docker, Docker Compose, GitHub Actions, deployment guide, and observability artifacts  
✅ Downloadable project package for every run  

This project helped me explore how agentic AI can support practical software engineering workflows with validation, review, and deployment-readiness built in.
